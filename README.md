# mnemosyne-mcp

![Mnemosyne MCP](https://github.com/user-attachments/assets/ebf5b04e-5270-40a1-b64e-686a7c2b87bd)

Mnemosyne MCP is a knowledge and memory server built around Airflow pipelines that ingest documents and web data, enrich and structure information, and maintain a persistent, shared knowledge layer for AI clients and agents to retrieve consistent context over time.
